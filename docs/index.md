## Scope

Digital Kiosk currently ships with:

- un frontend admin React + Vite,
- un player web avec fallback offline,
- une API Express avec SQLite,
- un shell Electron admin qui embarque le serveur,
- un player Electron Windows dédié en plein écran,
- une app Flutter pour la gestion mobile et desktop.
