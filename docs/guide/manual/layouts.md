# Manuel utilisateur — Layouts

## Objectif

Concevoir la structure visuelle affichée par les players.

## Actions

- créer un layout,
- définir les zones (texte/média),
- choisir un mode (`standard`, `room-door-display`, `room-status-board`),
- associer le layout aux écrans.

## Vérifications avant publication

- lisibilité sur écran cible,
- compatibilité des zones média,
- test avec données réelles.

## Erreurs fréquentes

- **Aucun affichage**: layout non assigné.
- **Contenu coupé**: dimensions de zones non adaptées.
