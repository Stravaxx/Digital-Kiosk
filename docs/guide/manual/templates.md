# Manuel utilisateur — Templates

## Objectif

Réutiliser des modèles d’affichage prêts à l’emploi.

## Utilisation

1. Choisir un template adapté au besoin.
2. Personnaliser textes/couleurs/zones.
3. Sauvegarder comme layout final.

## Recommandations

- privilégier la cohérence visuelle par site,
- tester mode classique et mode accessibilité si utilisé.

## Erreurs fréquentes

- **Rendu inattendu**: paramètres hérités non ajustés.
- **Template inadapté**: choisir un modèle selon contexte (board/door/standard).
