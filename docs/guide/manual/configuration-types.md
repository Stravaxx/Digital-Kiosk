# Manuel utilisateur — Types de configuration

## 1) Configuration initiale

- création du premier compte admin,
- vérification de la connectivité API,
- paramétrage minimum du système.

## 2) Configuration écrans/player

- pairing PIN/QR,
- assignation écran ↔ layout,
- rotation token si besoin.

## 3) Configuration contenu

- création assets,
- création playlists,
- mise en place layouts/templates.

## 4) Configuration planning

- salles,
- calendriers,
- validation du rendu board/door.

## 5) Configuration exploitation

- seuils alertes,
- politique storage,
- cadence de supervision Fleet/Ops.

## 6) Configuration sécurité

- gestion comptes/roles,
- politique sessions,
- journalisation et audit.

## Checklist finale

- au moins un écran `online`,
- un layout publié,
- une playlist fonctionnelle,
- alertes configurées,
- docs internes à jour.
