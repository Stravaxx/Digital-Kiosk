# Manuel utilisateur — Logs

## Objectif

Analyser les événements techniques et métiers du système.

## Utilisation

- filtrer par type/niveau,
- rechercher une période,
- corréler avec Fleet/Alerts.

## Cas d’usage

- incident player offline,
- échec de connexion admin,
- échec upload/import média.

## Erreurs fréquentes

- **Aucun log visible**: filtre trop restrictif.
- **Volume élevé**: ajuster politique de rétention.
