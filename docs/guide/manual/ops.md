# Manuel utilisateur — Ops

## Objectif

Suivre les indicateurs d’exploitation et SLA.

## Mesures principales

- disponibilité,
- incidents sur 24h,
- MTTR simplifié.

## Utilisation

- lecture quotidienne des KPI,
- comparaison avant/après changements,
- communication des indicateurs aux équipes.

## Erreurs fréquentes

- **KPI incohérents**: données incomplètes ou incident en cours.
- **SLA dégradé**: multiplier les vérifications Fleet + Logs.
