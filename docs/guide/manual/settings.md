# Manuel utilisateur — Settings

## Objectif

Configurer les paramètres globaux: sécurité, stockage, alertes et système.

## Zones clés

- authentification/admin,
- politique de stockage,
- seuils d’alertes,
- paramètres système.

## Bonnes pratiques

- documenter toute modification de seuil,
- appliquer les changements en fenêtre maîtrisée,
- vérifier l’impact dans Fleet/Alerts après modification.

## Erreurs fréquentes

- **Seuils trop stricts**: bruit d’alertes.
- **Seuils trop larges**: incidents détectés trop tard.
