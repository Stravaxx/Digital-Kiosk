# Manuel utilisateur — Assets

## Objectif

Centraliser les fichiers médias utilisés dans les playlists.

## Actions

- upload de fichiers,
- import, suppression, consultation,
- contrôle de taille et type.

## Formats recommandés

- images web optimisées,
- vidéos compressées (taille maîtrisée),
- noms de fichiers explicites.

## Bonnes pratiques

- nettoyer les assets non utilisés,
- utiliser une convention de nommage,
- vérifier l’espace disque avant imports massifs.

## Erreurs fréquentes

- **Upload refusé**: quota/policy dépassé.
- **Asset introuvable**: suppression antérieure ou ID invalide.
