# Manuel utilisateur — Playlists

## Objectif

Composer les séquences de contenus diffusées par les zones média.

## Utilisation

1. Créer une playlist.
2. Ajouter des items (images, vidéos, iframe, markdown).
3. Définir l’ordre et la durée.
4. Associer la playlist à une zone d’un layout.

## Conseils

- commencer par une playlist test courte,
- vérifier les durées image pour lisibilité,
- éviter les médias trop lourds.

## Erreurs fréquentes

- **Playlist vide**: aucun item actif après filtrage planifié.
- **Lecture interrompue**: média inaccessible ou corrompu.
