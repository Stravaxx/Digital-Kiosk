# Manuel utilisateur — Calendar

## Objectif

Synchroniser et contrôler les événements diffusés sur les écrans.

## Fonctions

- visualisation des événements,
- import/synchronisation des sources calendrier,
- contrôle des créneaux.

## Workflow conseillé

1. Vérifier les salles.
2. Vérifier les sources calendrier.
3. Contrôler les événements du jour.
4. Vérifier le rendu player (board/door).

## Erreurs fréquentes

- **Aucun événement**: source vide/inaccessible.
- **Horaires décalés**: fuseau horaire ou heure machine incorrecte.
