# Manuel utilisateur — Rooms

## Objectif

Administrer les salles utilisées par les écrans et le calendrier.

## Actions disponibles

- créer/modifier/supprimer une salle,
- définir nom, numéro, localisation,
- associer une salle à un écran.

## Recommandations

- standardiser les noms (ex: `Salle A-101`),
- renseigner la localisation pour l’affichage door/board,
- éviter les doublons de numéros.

## Erreurs fréquentes

- **Salle non visible sur player**: vérifier affectation écran/layout.
- **Conflit de libellé**: harmoniser naming dans toute l’organisation.
