# Installation

## Prérequis

- Node.js 20+
- npm 10+
- Git (recommandé)

## Développement

```bash
npm install
npm run dev
```

Cette commande démarre le système complet en local:

- API
- Frontend admin
- Player (route `/player`)
- Documentation complète + API docs

## Build application

```bash
npm run build
```

## Production complète

```bash
npm run prod
```

Cette commande initialise aussi la documentation automatiquement.

## Alias tout-en-un

```bash
npm run dev:all
npm run prod:all
```

## Générer la documentation complète

```bash
npm run docs:build
```

## Prévisualiser la documentation

```bash
npm run docs:preview
```

## Documentation API seule

```bash
npm run docs:api
```
